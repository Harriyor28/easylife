# Easy Life Electrician Consult

Professional electrical and solar energy services website for residential, commercial, and industrial clients across Nigeria.

## Features

- Responsive landing page with services, gallery, testimonials, and contact sections
- Admin control panel (`admin.html`) for managing content in real-time
- Dynamic service and testimonial management via localStorage
- Customizable branding (logo, colors) through the admin panel

## How to Use

1. Clone the repo and open `index.html` in a browser.
2. For admin access, open `admin.html` and log in with password `admin123`.
3. Add/edit services and testimonials from the admin panel — changes appear on the main site in real time.

## Deploy

This is a static site — deploy to **Vercel** or **Netlify** by connecting this repository. No build step required.

## Project Structure

- `index.html` — Main website
- `styles.css` — Website styling
- `script.js` — Website JavaScript + admin data integration
- `admin.html` — Admin control panel
- `admin-styles.css` — Admin panel styling
- `admin-script.js` — Admin panel logic
- `ADMIN_PANEL_README.md` — Admin panel usage guide
- `INTEGRATION_GUIDE.md` — Integration documentation
- `INTEGRATION_COMPLETE.md` — Integration completion report
